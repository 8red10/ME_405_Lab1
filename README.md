## ME 405 Lab 1

### Authors
By: Jack Krammer, Jason Chang

California Polytechnic State University

February 6, 2024


### Description
This project was aimed to increase our familiarity with pulse width modulation (PWM).
In this lab, we first applied PWM signals to change the brightness on an LED. Then, 
we applied PWM to drive motors using a L6206 H-bridge.
