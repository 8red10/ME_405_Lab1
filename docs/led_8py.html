◆ led_brightness()

led.led_brightness ( channel,
duty )

Sets the duty cycle of the PWM on the input channel to the input level.

If the input duty cycle is less than 0, the duty cycle is set to 0. If the input duty cycle is greater than 100, the duty cycle is set to 100.

Parameters
channel-> A TimerChannel object to set the duty cycle on.
duty-> The duty cycle to set the PWM to.
Returns
None.

led.main ( )

Sets up a PWM output on pin PA0 where the LED is and ramps the duty cycle from 0 to 100 over a 5 second period (in 50 millisecond increments).

Loops this behavior forever. This main code is run if this file is the main program but won't run if this file is imported as a module by some other program.
